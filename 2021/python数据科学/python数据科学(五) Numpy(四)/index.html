3 | 数组的排序

np.sort()提供快速排序, 在不修改原始数组的基础上返回一个排好序的数组

np.argsort()返回原始数组排好序的索引值

np.partition(x, K)返回新数组, 左边是数组中最小的K个值, 右边是原数组的其他值

同样地, np.argpartition()函数计算的是分隔的索引值

可以通过指定axis参数对行/列排序 (这样处理将行列当成独立的数组, 丢失行列值之间的关系)
