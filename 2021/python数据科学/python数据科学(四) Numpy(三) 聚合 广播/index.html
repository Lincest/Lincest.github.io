广播的规则

  1. 如果两个数组维度数不同, 小维度数组的形状将在左边补1
  2. 如果两个数组的形状在任何一个维度都不匹配, 数组的形状会沿着维度为1的维度扩展以匹配另外一个数组的形状
  3. 如果两个数组的形状在任何一个维度都不匹配并且没有任何一个维度等于1, 则会引发异常
