2025: 我的 AI 折腾日记 (不定期更新)

本文最后更新于 2025年3月2日 下午

AI / Chatbot 工具更新速度越来越快, 打算将 2025 年自己的 AI stack 更新在这个 Blog 下, 本文不定期更新, 主要为了好玩而记录, 同时可以和大家交流一下, 保持一点追求潮流的热情~

Changelog

2025-02-02

  • 开通了 perplexity education pro (闲鱼花 30 买的到年底的会员)
    • 试用了一下 perplexity + o3-mini 和 perplexity + deepseek r1，体验非常不错。
    • APP 感觉也优化了，比一个月前用的丝滑。

2025-02-01

  • 体验了一下 gpt-o3-mini, 感觉还行, 不过官网还是会因为 IP 原因降智, 就因为这个也不会选择开 closeai 的 plus

2025-01-31

  • 使用小红卡充值了 3 美元的 Perplexity, 开通了 API, 目前使用方式:
    1. 使用 raycast 的 Perplexity API 插件
    2. 配合 chatbox 使用 llama-3.1-sonar-large-128k-online 模型
  • 申请了 NVIDIA API, 可以免费使用 deepseek r1, 不过目前响应速度还很慢, 加到了 chatbox 备用

2025-01-27

  • 在闲鱼上花了 30 订阅了 2 个月的 cursor pro，具有 5000 额度，不知道能用多久。
    • 计划是用来写之后的文档和论文，并辅助用于开发一些自己的小工具、script，以及优化一些workflow
  • 将 chatbox 配置更多 API，用分 session + 置顶的方式（ref）绕过 API 只能配置一个的问题。

2025-01-22

  • 重新下载使用了 chatbox
    • 更新很及时, 隔壁的 nextchat 上次发布 docker image 已经是 2 个月前了.
    • 支持的 artifacts 和 联网搜索还是挺好用的
    • 但是不支持数据同步, 有点可惜

2025-01-20

  • 开始体验 deepseek r1，之前没有仔细使用过 r1-lite，不过感觉 r1 还是很不错的，问了几个 o1-mini 无法回答的比较刁钻的问题都能回答正确。
  • 看到 deepseek 还开放了 r1 api：deepseek-reasoner，打算之后接入到自己的 nextchat，如果体验良好之后就慢慢抛弃 gpt 和 claude 的每月订阅，全面拥抱 deepseek 了。

2025-01-17

  • 使用 obsidian 的一个 copilot AI 插件, 整体体验非常不错, 可以在上百篇笔记中比较准确的搜索出想要的问题的答案, 以及自动关联相关笔记.
    • Embedding 使用 siliconflow (硅基流动) 的 BAAI/bge-m3
    • Chat 使用 deepseek-v3 API

2025-01-11

  • 重新订阅 Claude Pro (20$ / 月), 并通过 fuclaude 的方式与两个朋友合租.
  • 开通 oaipro 并充值了 10$, 自建了 NextChat 来使用 claude, gpt-4o, gemini 和 deepseek 的 API, 作为 claude / openai 官网不可用时的备选方案.
  • 下载了 deepseek 的 ios 移动端, 体验良好

2025-01-09

  • 开始使用 deepseek-v3, 在半年前使用豆包和kimi后很久没碰过国产模型了, 重新用起来感觉比以前强大了太多, 搜索和 deepthink 都很给力, 除了有些敏感问题无法询问, 没什么很大的缺点

2025-01-01

  • 开始探索 cursor , 并在之后的几天用 cursor 写了 N 多个自用的小脚本和服务, 总结了一些 cursor 的用法

2024 年末

  • 主力使用的 chatbot 是 gpt-4o (合租), 感觉 gpt 经常遇到故障和降智, 因此开始探索别的模型, 中间开过一个月的 Claude Pro, 体验不错, 但是不带 Search 功能比较遗憾, 并且因为价格比较贵没有继续续费.
  • 在 Gemini 重磅更新后, 辅助使用 Gemini exp 1206
  • 同时使用 perplexity pro 进行一些基本问题的搜索.
